## **# Employee Wellness \& Productivity Analytics**



A full data analytics project using SQL, Python, and Power BI to understand employee well-being, burnout risk, productivity patterns, and key drivers of performance.



#### **## Project Structure**



Employee\_Wellness\_Project/

├── data\_raw/ # Original dataset

├── data\_processed/ # Processed datasets for BI

├── python/ # Python scripts + requirements

├── sql/ # PostgreSQL setup + queries

├── powerbi/ # Dashboard file

├── visuals/ # All charts generated

└── documentation/ # Report, data dictionary, README





#### **## Technologies Used**

\- Python (Pandas, Seaborn, Scikit-Learn)

\- PostgreSQL

\- Power BI

\- VS Code



#### **## Key Insights**

\- Stress level and sleep hours strongly influence burnout.

\- Manager support is a strong indicator of well-being.
